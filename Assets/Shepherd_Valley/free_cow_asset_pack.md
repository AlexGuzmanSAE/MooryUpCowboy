# Free Cow Asset Pack Documentation

## Overview

This **Free Cow Asset Pack** provides a stylized, game-ready 3D cow model optimized for real-time use in engines like Unity. It includes multiple animations and is suitable for farming, simulation, or stylized adventure games.

---

## Model Details

- **Vertices:** 2,472  
- **Triangles:** 3,814  
- **UV Mapping:** Yes (UV1)  
- **Vertex Colors:** Included  
- **Skin Weights:** Included (rigged for animation)  

The model is lightweight and optimized for performance while maintaining a clean stylized look.

---

## Animations

The cow comes with the following animations:

- **Eating** (`A_Cow_Eating_01`)
- **Idle** (`A_Cow_Idle_01`)
- **Idle Break** (`A_Cow_IdleBreak_01`)
- **Run** (`A_Cow_Run_01`)
- **Walk** (`A_Cow_Walk_01`)

All animations are loopable and suitable for blending inside an Animator Controller.

## Usage

### Import

Import the asset files into your project.

### Setup

1. Drag the cow model into your scene  
2. Assign or create an Animator Controller  
3. Add animation states (Idle, Walk, Run, etc.)  
4. Configure transitions as needed
